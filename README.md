Установка GPIO
Установить LGPIO легко! Убедитесь, что ваши репозитории программного обеспечения обновлены:

sudo apt update
И установите библиотеку Python LGPIO с помощью:

sudo apt install python3-lgpio
Сделанный. Вы готовы идти.
